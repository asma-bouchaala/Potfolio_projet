# Portfolio Web Personnel

## Contexte & Objectif
Développement d’un site web personnel type portfolio pour présenter mon parcours académique, mes compétences, mes projets et activités. Le site est pensé pour être clair, structuré et professionnel, offrant un outil efficace pour valoriser mon profil .
## Structure & Technologies
- **HTML5 / CSS3 / JavaScript** : structure, style et interactivité.  
- **Organisation** : pages séparées pour Accueil, Parcours, Compétences, Projets, Stages & Expériences, Événements, Passions, Témoignages, Quiz, Contact.  
- **Feuille de style unique** : `style.css` pour cohérence graphique.  
- **Ressources** : dossier `images/` pour tous les visuels.  
- **Quiz interactif** : JavaScript gère les scores et affiche les reponses correctes.

## Fonctionnalités principales
- **Navigation fluide** : menu global sur toutes les pages.  
- **Pages interactives** : quiz avec score et corrections détaillées.  
- **Formulaire contact fonctionnel** : nom, email, sujet et message.  


**Technologies utilisées :** HTML5, CSS3, JavaScript.
